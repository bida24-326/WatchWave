About WatchWave

WatchWave is a student-developed website by Oratile Hikhwa as part of the Web & Multimedia Development course at Botswana Accountancy College. This platform showcases a functional and visually appealing space for browsing handpicked Netflix movies and TV shows.

We aim to make entertainment discovery easier, more engaging, and visually fun! Our collection highlights fan-favorite shows and films along with quick descriptions, giving you just enough to pick your next binge-worthy title.

Tools and Technologies Used

  • HTML5 for clean structure
  • CSS3 for animations and layout styling
  • JavaScript for dynamic interaction
  • GitHub Pages for live deployment
  • Visual Studio Code for development
  • GIMP for image editing

Project Goals

  • Deliver a modern and accessible user experience
  • Ensure mobile-friendly and responsive design
  • Showcase coding proficiency and creativity
  • Encourage content discovery through aesthetics
