About WatchWave

WatchWave is a student-developed website by Oratile Hikhwa as part of the Web & Multimedia Development course at Botswana Accountancy College. This site is designed to showcase a creative and functional platform for browsing curated Netflix movies and TV shows.

The aim of WatchWave is to provide users with quick, engaging overviews of popular entertainment content. Each movie and series is presented with its rating, genre, and synopsis, to help viewers find something they'll love.

Tools and Technologies Used

  • HTML5 for structure
  • CSS3 for styling and layout
  • JavaScript for interactivity
  • GitHub Pages for deployment
  • Visual Studio Code as IDE
  • GIMP for image editing

Project Goals

  • Create a visually appealing, user-friendly website
  • Ensure responsiveness across devices
  • Demonstrate competency in web development techniques
  • Promote creativity in media presentation
